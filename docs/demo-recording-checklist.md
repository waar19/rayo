# Demo Recording Checklist (GIF/video source)

Use this sequence to capture short demo media for README/release pages.

## Scene 1: GUI search speed

1. Open `rayo-gui`.
2. Type query with fast results (`report`, `ticket`).
3. Show action buttons (`Open`, `Open as admin`, `Open folder`).

## Scene 2: PowerToys Run integration

1. Open PowerToys Run.
2. Run `ry report`.
3. Show result open action.

## Scene 3: Update/install path

1. Terminal:
   - `winget install waar19.Rayo`
2. Show GUI update banner (when a newer release exists).

## Output targets

- README asset: short GIF (8-15 seconds).
- Release page asset: MP4 clip (optional, higher quality).
